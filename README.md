# Vue 3 + Typescript + Vite

This template should help get you started developing with Vue 3 and Typescript in Vite.

## Features

- 🚀 vue3
- 🚀 Vite
- 🚀 Typescript
- 🍭 Pinia 状态管理
- 🍭 Axios 封装及接口管理
- 🍭 Eslint+Pettier 统一开发规范
- 🍭 Commitizen + Commitlint 提交规范
- 🍭 husky + lint-staged Git Hook工具
- 🍭 Element Plus UI 框架
